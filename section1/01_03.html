Анатомия CSS

CSS контролирует все аспекты дизайна веб-старницы

  1. шрифты
  2. размеры
  3. цвета
  4. позицию
  5. макет
  6. расстоние
  7. анимацию
  8. и другое

CSS создаёт правила стилазиации. Простейшее правило состоит из селектора и объявления в фигурных скобах.

Пример объявления color: green

Объявление состоит из 2 частей: property и value

Выводы урока

  1. Код CSS состоит из правил стилизации
  2. Правило состоит из селектора и набора деклараций (объявлений)
  3. Декларация состоит из пары свойство: значение
